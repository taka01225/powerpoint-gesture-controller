●ハンドサインによるアプリケーション操作制御プログラム

〇概要
WebカメラとMediaPipeを用いて手の骨格を検出し、特定のハンドサイン（人差し指を曲げる動作）でアプリケーションなどのPC画面を自動操作するプログラムです。
大学の卒業研究の一部として作成したものになります。

〇主な機能
・リアルタイムな手の骨格検出: Webカメラの映像から「MediaPipe」を用いて手のランドマーク（関節などの特徴点）を高精度に検出します。
・ハンドサイン判定: 人差し指の指先（第1関節）と第2関節のY座標を比較し、「人差し指を曲げた」という動作をトリガーとして認識します。
・PCの自動操作:ハンドサインを認識すると「PyAutoGUI」を用いてマウスカーソルを自動制御し、PowerPointの起動、新しいプレゼンテーションの作成、画像の挿入といった一連の動作を自動で行います。

〇開発環境・使用ライブラリ
・Python 3.9
・Anaconda（仮想環境によるPython 3.9環境の構築）
・Spyder（コード編集・実行のための統合開発環境）
・OpenCV（opencv-python）
・MediaPipe （mediapipe==0.10.14）
・PyAutoGUI （pyautogui）
・Pyperclip （pyperclip）

〇ファイル構成
・main.py: カメラの起動、ハンドサインの検出、およびアプリケーションの自動操作を行うメインプログラムです。
・get_position.py: 使用するPCの画面解像度に合わせて、クリックさせたい場所（アイコンなど）のX座標・Y座標を調べるための補助ツールです。

〇使い方
本プログラムは実行するPCの画面サイズやアイコンの配置に合わせて座標を設定する必要があります。

1. 座標の取得:get_position.pyを実行し、3秒以内に調べたい場所（PowerPointのアイコン等）にマウスカーソルを合わせます。コンソールに出力されたxとyの数値をメモします。
2. 座標の設定:main.pyの後半にあるpg.doubleClick(x=..., y=...)などの数値を手順1で取得したご自身のPCの座標に書き換えます。
3. 実行:main.pyを実行するとWebカメラが起動します。カメラの前で人差し指を曲げると設定した座標に沿ってマウスが自動で動きPowerPointの操作を開始します。終了する場合はカメラ映像のウィンドウを選択してqキーを押します。

〇今後の展望
現在は座標を直接コードに記述していますが今後は画像認識（PyAutoGUIの画像検索機能）を活用し、どんなPC環境でも事前の座標設定なしに自動でアイコンを探してクリックできる汎用性の高いシステムへの改良を検討しています。
